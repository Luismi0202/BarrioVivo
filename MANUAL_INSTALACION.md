# Manual de instalación - BarrioVivo (APK)

Este manual explica cómo instalar BarrioVivo en un móvil Android usando el release del proyecto. En GitHub hay un release con una carpeta comprimida que dentro lleva el APK.

La instalación básica es: descargar el release, descomprimir, abrir el APK y darle a instalar.

## 1. Qué necesitas

- Un móvil Android.
- Conexión a internet para descargar el release (si lo haces desde el propio móvil).
- Un gestor de archivos para localizar la descarga y abrir el APK.
  - Normalmente Android ya trae uno (Archivos / Files), pero si no, cualquiera de Play Store te sirve.
- Para descomprimir:
  - Si el release está en .zip: normalmente Android lo descomprime desde el propio gestor de archivos.
  - Si el release está en .rar o similares: necesitas una app tipo RAR o ZArchiver.

## 2. Instalación desde el móvil (descargando el release)

### 2.1 Descargar el release

1. Entra al repositorio de GitHub desde el navegador del móvil.
2. Ve a la sección Releases.
3. En el último release, descarga la carpeta comprimida (zip/rar) donde viene el APK.

3. Busca el archivo comprimido del release.

### 2.3 Instalar el APK

1. Dentro de la carpeta extraída, localiza el archivo .apk.
2. Pulsa el APK.
3. Android te va a pedir confirmación.
4. Si te sale el aviso de "instalar apps desconocidas", tienes que permitirlo para tu navegador o para tu gestor de archivos.
5. Dale a instalar.

## 3. Instalación alternativa: descargar en PC y pasar al móvil

Si desde el móvil te da problemas descargar o descomprimir, lo puedes hacer en PC y pasarlo.

### Opción A: pasar por USB

1. Descarga el release en el PC.
2. Descomprime la carpeta en el PC.
3. Conecta el móvil por USB.
4. Copia el APK a una carpeta fácil (por ejemplo Descargas).
5. En el móvil, abre el gestor de archivos y busca el APK.
6. Pulsa el APK y dale a instalar.

### Opción B: pasarlo por WhatsApp

1. Descarga el release en el PC.
2. Descomprime.
3. Entra al WhatsApp Web.
4. Mándate el APK a tu propio número (mensaje personal).
5. En el móvil descargas el APK desde WhatsApp.
6. Lo abres desde Descargas y lo instalas.

Nota: a veces WhatsApp limita algunos tipos de archivo, si no te deja mandar el .apk directamente, lo puedes mandar comprimido en .zip.

## 4. Problemas típicos

### 4.1 No me deja instalar por seguridad

Normal: Android bloquea por defecto la instalación desde fuera de Play Store.

Solución:

- Cuando te salga el aviso, permite "Instalar apps desconocidas" para la app con la que estás abriendo el APK (Chrome, Files, etc.).

### 4.2 No encuentro el archivo

- Mira en Descargas.
- Si lo bajaste desde el navegador, revisa el historial de descargas.
- Si lo recibiste por WhatsApp, normalmente está en la carpeta de WhatsApp/Media o en Descargas.

### 4.3 No puedo descomprimir

- Si es .zip normalmente no hay problema.
- Si es .rar, instala una app que soporte descompresión.

## 5. Verificación

Una vez instalada, abre la app y comprueba:

- Que arranca y sale la pantalla de login.
- Que puedes registrarte.
